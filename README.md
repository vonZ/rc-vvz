# vvz-react-component

A library with re-usable react components

## Install
```
npm install vvz-react
```
## Scripts
Update the docs:
```
predeploy:docs then deploy:docs
```
Publsh new to npm:
```
prepublish then inside lib-folder: npm publish
```
